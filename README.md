# Pali Coin

A secure cryptocurrency implemented in Rust with advanced security features.

## Features

- ChaCha20-Poly1305 wallet encryption
- BIP39 seed phrases for backup and recovery
- ECDSA signatures for transaction security
- Mining with adjustable difficulty
- Web interface for wallet management
